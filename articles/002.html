// The web is Bloated

Today i was using the web and noticed some sites only use 10KB and some used 2MB, which is ridiculus if a fully functional site that adds many features like youtube can be 2KB why cant youtube itself be 2KB, even so the 2KB site has more features?

You might say its because youtube is better, it has more features but this isn't true. Other factors may be bandwidth but the average internet speed around the globe is only 1MB/s

How can we fix this?

First we should start by making use of CSS and HTML only to remove most of bloat which is Javascript, then we have to educate the Developers on bloat so they can make better web apps and websites for their users, so remote areas can access critical and not critical services via the internet.
